**Improvements from Parent**

**Key Issues**
- Cosine LR `lr_lambda` function does not clamp `progress` at 1.0, causing unintended rebound into second cycle after epoch 80; best epoch 114 coincides with ascending ramp of this second cycle
- Stronger regularization successfully delayed best epoch from 31 (parent) to 114 and slowed memorization, but test F1 improvement is marginal (+0.0006)
- Primary bottleneck is the **representational ceiling of frozen STRING_GNN 256-dim embeddings** — input features are fixed and informationally bounded, limiting gains from head architecture changes
